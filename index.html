¿Qué es Git?

Git es un Software de control de versiones distribuido que permite a un equipo de desarrolladores trabajar sobre el mismo código.

Control de versiones distribuido

Se denomina "distribuido" porque cada miembro del equipo dispone de una copia completa del código.

Vetajas de usar Git

  • Versionado de código
  • Código colaborativo
  • Recuperación de archivos
